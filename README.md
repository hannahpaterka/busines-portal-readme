## Overview

A business portal for **vendors, dealers, contractors, and office staff** — secure role-based access, centralized records, and tools for coordinating field and back-office work.

I built this for a contract client and currently maintain the Heroku production environment.

> **Production ready** — responsive across desktop and mobile, backed by PostgreSQL, and deployed on Heroku.

## Features

|  | Capability | What it does |
| :---: | --- | --- |
| 🔐 | **Users & access** | Role-based login and management for installers, dealers, and vendors |
| 📋 | **Work orders** | Create, assign, update, and track every stage of a job |
| 🧾 | **Invoicing** | Build and monitor invoices through a dedicated workflow |
| 📍 | **Field tools** | Maps, tap-to-call numbers, and quick email links for mobile teams |
| 💬 | **Team notes** | Keep vendors, contractors, and office staff aligned |
| 🔎 | **Fast search** | Find and filter PostgreSQL records without slowing down |
| 📱 | **Responsive UI** | Work comfortably from desktop, tablet, or mobile |
